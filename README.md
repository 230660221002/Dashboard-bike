# Bike Sharing Data Analysis

Proyek ini bertujuan menganalisis pola penyewaan sepeda berdasarkan musim, cuaca, dan waktu.

## Struktur Folder
- notebook.ipynb → Proses analisis lengkap
- data/ → Dataset asli
- dashboard/ → Aplikasi Streamlit
- requirements.txt → Dependencies

## Cara Menjalankan Dashboard
1. Install requirements
2. Masuk ke folder dashboard
3. Jalankan: streamlit run dashboard.py
